Biopharm Roulette

A for-fun project that webscrapes biopharmacy stocks with upcoming catalyst events from biopharmcatalyst.com using Selenium, then short sells those stocks using Alpaca's Trading API if deemed eligible to be short sold. Only stocks that are in phase 2 of clinical testing and within 3 days of the current date will be short sold.

Skills: Python, API, Webscraping
